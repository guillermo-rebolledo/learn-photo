The Source Photographs keep identical color rendering in both themes. Desktop keeps photograph, Histogram, and controls in one cause-and-effect surface; phone layout places them in that same order without horizontal overflow. Reduced motion replaces the 220 ms shutter sweep through the repository's existing stationary-dimming rule.
